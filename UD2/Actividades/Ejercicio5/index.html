UD1: Ejercicio 2

Realiza un script con if y otro con switch que permita introducir el precio de un producto mediante la función prompt y muestre un mensaje dependiendo del rango de precios.

  • Si el precio es menor de 10, se mostrará el mensaje, "Producto muy barato".
  • Si el precio está entre 10 y 29 (inclusive), se mostrar el mensaje, "Producto económico"".
  • Si el precio está entre 30 y 59 ambos inclusive, se mostrará el mensaje, "Producto de gama media".
  • Si el precio está entre 60 y 99 ambos inclusive, se mostrará el mensaje, "Producto costoso".
  • Si el precio es más de 100 o más, se mostrará el mensaje, "Producto de lujo".
  • Si la entrada no es un número válido, mostrar un mensaje "Error, debe ser un número válido"
